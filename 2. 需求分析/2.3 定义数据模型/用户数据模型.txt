**用户数据模型**

| 字段名      | 数据类型   | 描述     | 是否主键 | 是否可为空 |
|------------|------------|----------|----------|------------|
| userId     | INT        | 用户ID   | 是       | 否         |
| username   | VARCHAR(50)| 用户名   | 否       | 否         |
| password   | VARCHAR(255)| 密码   | 否       | 否         |
| email      | VARCHAR(100)| 邮箱地址 | 否       | 否         |
| phone      | VARCHAR(15)| 手机号码 | 否       | 否         |
| created_at | DATETIME   | 创建时间 | 否       | 否         |
| updated_at | DATETIME   | 更新时间 | 否       | 否         |

---

请注意，以上内容为示例用户数据模型，实际数据模型应根据具体需求进行设计。
